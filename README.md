# DualRun
Run multiple single-session app accounts at once. Multiple Viber, Discord, Steam, Snapchat accounts using one app on the same user on one pc/laptop (Windows). Open same app from two different accounts using two windows: original app's window and the DualRun script window. Not a complex software. README.md for guide and for more technical details.

# **Last update: Only Viber duper feature is tested** *11/05/2026*


## Documentation
Not all modern apps or social media support multiple  
 simultaneous sessions on the same device and the same user. Here I am talking about Windows. On phones it's easier to use third-party apps to clone app, or even built-in features like *Secure folder* on Samsung.
Apps like Telegram, Viber, Steam and a lot more  
 are installed only once per device. It means, only one directory can be created for an app, say (C:\Users\AppData\Local\Viber). This is not an issue that will be solved once and forever efficiently by app developers. And it is for a couple of reasons. One example I can think of is a Singleton rule, which stands for "don't start the other instance, if a one is already running". The issue is efficiency, as if the same instance is running - it devours twice as much RAM and CPU resources. Security is also one of reasons, preventing session token theft (hijacking). So to say - not convenient, but simple solution for many problems, get used to it. Why Steam does not allow "dualrun" answers itself.  
 So what is the reason this application is created  
  for? Often in a corporate environment fast access to the Viber or Telegram of a manager on a business trip is key for speed, customer satisfaction and productive work maintenance.
